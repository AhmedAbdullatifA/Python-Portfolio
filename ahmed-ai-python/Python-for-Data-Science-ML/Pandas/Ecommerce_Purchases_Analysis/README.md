# Ecommerce Purchases Analysis 🛒📊

Exploratory Data Analysis (EDA) project on an Ecommerce Purchases dataset using **Python & Pandas**.

## 🔧 Tools & Libraries
- Python
- Pandas

## 📁 Dataset
- **EcommercePurchases.csv**
- Contains customer purchase data including:
  - Address
  - Job
  - Language
  - Purchase Price
  - Credit Card details
  - Email

## 🧠 Analysis Includes
- Dataset inspection (head, shape)
- Descriptive statistics
- Filtering & conditional analysis
- Job title frequency analysis
- Time-based analysis (AM / PM purchases)
- Credit card provider analysis
- String manipulation (emails, expiration dates)
- Advanced filtering with multiple conditions

## 📌 Key Questions Answered
- What is the average, highest, and lowest purchase price?
- How many users prefer English on the website?
- What are the most common job titles?
- How many purchases were made in AM vs PM?
- Which credit cards expire in 2025?
- What are the most popular email providers?
