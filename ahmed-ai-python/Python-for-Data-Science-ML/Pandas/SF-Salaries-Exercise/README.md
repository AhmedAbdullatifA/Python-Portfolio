# SF Salaries Exercise 📊

Exploratory Data Analysis (EDA) project using the San Francisco Salaries dataset.

## 🔧 Tools
- Python
- Pandas

## 📁 Dataset
- **Salaries.csv** – San Francisco public employee salaries

## 🧠 Analysis Includes
- Dataset inspection (head, info)
- Handling missing values
- Salary statistics (BasePay, OvertimePay, TotalPayBenefits)
- Job title analysis
- Grouping & aggregation by year
- String operations
- Correlation analysis

## 📌 Key Questions Answered
- What is the average BasePay?
- Who is the highest and lowest paid employee?
- What are the most common job titles?
- Is there a correlation between job title length and salary?
